    (2)

    Mark - таск менеджер

    для контроля списка задач

    Отображает дату и хранит список добавленных задач с их статусом. Есть встроенный таймер Pomodoro для лучшей фокусировки во время работы
    Присутствует возможность изменения данных пользователя и настройки внешнего вида
    Оптимизирован для всех типов устройств

    Написан на чистом JavaScript. Вёрстка - HTML5 и CSS3

    (3)

    Клиент-серверное приложение

    для тестирования знаний студентов

    Главная функция приложения - создание тестов с несколькими вариантами ответов и их последующее прохождение
    Настроена возможность авторизации двух типов пользователей: администраторов и студентов, обладающих разными правами доступа к функциям
    Администраторы могут создавать, удалять и редактировать тесты. Студенты - проходить их, отслеживая свой прогресс в личном кабинете (там представлена история тестирований с полученными баллами и данными ответами)

    Серверная часть реализована на PHP. В создании фронтенда задействован фреймворк Bootstrap.

    (4)

    Клиент-серверное приложение

    для оформления заявок на кредит

    Настроены сценарии для авторизации пользователей, редактирования личных данных и удаления аккаунта
    Далее - создание заявое на кредиты разных категорий

    Серверная часть написана на Node.js. Для хранения данных используется локальная БД посредством SQLite3.
    Фронтенд - JavaScript, HTML5 и CSS3. Задействован шаблонизатор EJS

    (5)

    для кодирования сообщений с помощью алгоритма Касуми

    Разработано для кодирования/декодирования сообщений, введенных или загруженных пользователем
